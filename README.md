# Power Plants Analysis in Germany

This project analyses conventional power plants in Germany in 2020 using open data source.

## Tools
Python, Pandas, Matplotlib

## Data source
https://data.open-power-system-data.org/conventional_power_plants/

## Steps
- Loaded the CSV dataset
- Aggregated installed capacity by energy source
- Created visualization
- Generated analytical insights
